# Username Validator

## Description
This program validates usernames according to specific rules. It prompts the user to enter a username and checks if it meets all the requirements.

## Username Rules
- Only lowercase letters, digits, and underscore characters are allowed
- Must start with a lowercase letter
- Must contain at least three but no more than 12 characters

## Usage
Run the script:
```bash
./username.sh
```

Or test with an input file:
```bash
./username.sh < username-input
```
